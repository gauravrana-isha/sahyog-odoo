# Sahyog Volunteer Guide

## Getting Started

### Login

1. Go to `https://isha.sahyog.online/sahyog/login`
2. Click **Login with Google** and sign in with your registered email
3. You'll be redirected to the Sahyog Volunteer Portal

If you see "No volunteer record linked to your account", contact your admin — your Google account needs to be linked to a volunteer profile.

### Dark Mode

Tap the moon/sun icon in the top-right header to toggle between light and dark mode. It follows your device preference by default.

---

## Navigation

The app has 4 tabs at the bottom (mobile) or a sidebar (desktop):

| Tab | What it does |
|---|---|
| **Programs** | Browse upcoming program schedules |
| **History** | View all your past and current entries |
| **Request** | Submit new requests (program, break, silence, unavailability) |
| **Calendar** | Team calendar showing everyone's entries |

**Profile** is accessible via the avatar icon in the top-right header.

---

## Programs Tab

Shows upcoming program schedules that are relevant to you:
- Only programs of type **Main** and **Hatha** are shown
- Programs you've already completed are hidden
- Programs restricted to a different gender are hidden
- Prerequisites of your completed programs are hidden

Each card shows: program name, dates, time window (if set), location, capacity, fee, and a **Recurring** badge if applicable.

**To enroll:** Tap the **Enroll** button on any program card. This takes you to the Request page with the program and schedule pre-filled.

**Search:** Use the search bar at the top to filter by program name or location.

---

## Request Tab

Submit 4 types of requests:

### Program Enrollment
1. Select **Program** tab
2. Choose a program from the dropdown
3. If schedules are available, pick one — dates, times, and location auto-fill
4. Choose participation type: Participant or Volunteer
5. Add optional notes
6. Tap **Submit Request**

Your request goes to admin for approval (status: Pending Approval).

### Break Request
1. Select **Break** tab
2. Choose break type: Personal, Health, or Family Emergency
3. Pick start and end dates (end must be >= start)
4. Add optional reason and notes
5. Tap **Submit Request**

### Silence Request
1. Select **Silence** tab
2. Choose silence type:
   - **Personal** — standard silence period
   - **9PM–9AM** — recurring daily silence, auto-sets times to 21:00–09:00
   - **Program** — silence linked to a specific program (select from dropdown)
3. Pick start and end dates
4. For recurring silence, check **Recurring** and set start/end times
5. Tap **Submit Request**

**9PM–9AM note:** The recurring checkbox and times are locked — they're always 21:00–09:00.

### Unavailability
1. Select **Unavailability** tab
2. Pick a date
3. Set start and end times (uses time picker, end must be after start)
4. Add optional reason
5. Tap **Mark Unavailable**

Unavailability is recorded immediately (no admin approval needed).

---

## History Tab

Shows all your entries across all types, sorted by date (newest first).

### Filters
Use the segmented control at the top to filter: All, Programs, Breaks, Silence, Unavailability.

### Entry Cards
Each card shows:
- Type badge (color-coded)
- Status badge (Approved, On Going, Done, Pending, etc.)
- Date range
- **Recurring** badge with time window (if applicable)
- Notes (expandable)

### Actions

**Cancel:** Entries with status Pending Approval or Approved show a **Cancel** button.

**Accept/Reject:** If an admin created an entry for you with status "Pending Confirmation", you'll see **Accept** and **Reject** buttons.

**Delete:** Unavailability slots have a delete (trash) icon.

### Status Meanings

| Status | Meaning |
|---|---|
| Pending Approval | Waiting for admin to approve |
| Pending Confirmation | Admin created it, waiting for you to accept |
| Approved | Approved, not yet started |
| On Going | Currently active |
| Done | Completed |
| Cancelled | Cancelled by you or admin |
| Dropped | Program enrollment was dropped |

---

## Calendar Tab

Shows a Gantt timeline of all team members' entries.

### Controls
- **Month/Week** toggle to change zoom level
- **Today** button to jump to current date
- **Previous/Next** arrows to navigate
- **Entry type filter** dropdown to show only specific types

### Colors
- Blue = Silence
- Orange = Break
- Green = Program
- Gray = Unavailability

The red vertical line marks today. The calendar auto-scrolls to today on load.

### Reading the Timeline
- Each row is a volunteer
- Colored bars show their entries spanning across dates
- Hover over a bar to see details (name, dates, status)

## Profile

Tap the avatar icon in the top-right header to open your profile.

### View Mode
Your profile is organized in accordion sections:
- **Contact** — email (read-only), phone, WhatsApp
- **Personal** — gender, birthday, nationality, city, state, region
- **Languages & Skills** — languages spoken, special skills
- **Health** — health conditions
- **Volunteer Info** — volunteer types, joining dates, added by
- **Emergency Contact** — name, phone, relation

### Edit Mode
Tap the blue floating edit button (bottom-right) to enter edit mode.

**Editable fields:**
- Phone, WhatsApp
- Nationality, City, State
- Languages (multi-select dropdown)
- Special Skills, Health Conditions
- Volunteer Types (multi-select dropdown)
- Emergency Contact (name, phone, relation)

**Read-only fields:** Name, Email, Gender, Birthday, Region, Work Mode, Sub Team, Role, Joining Dates

Tap **Save Changes** (sticky button at bottom) when done. Tap **Cancel** to discard changes.

### Photo
- Tap your avatar to view full-screen photo
- Tap the camera icon to upload a new photo

### Close Profile
Tap the **X** button in the top-right header. The bottom navigation is hidden while profile is open.

**Note:** You cannot close the profile while in edit mode — save or cancel first.

---

## Notifications

Tap the bell icon in the top-right header to open the notification drawer.

### What you'll receive notifications for:
- Silence/Break approved or cancelled
- Program enrollment (by admin)
- Meeting scheduled
- Request expired (auto-cancelled by system)

### Actions
- **Mark as Read** — tap on each notification
- **View** — some notifications have a blue "View" link that takes you to the relevant history entry

Unread count shows as a red badge on the bell icon. It refreshes every 60 seconds.

---

## Tips

- **Overlap protection:** You can't have overlapping silence and break periods. The system will show an error if you try.
- **Silence limits:** If your annual silence days exceed your volunteer type's maximum, you'll see a warning (but the request still goes through).
- **Schedule pre-fill:** When enrolling via the Programs tab, dates and location auto-fill from the selected schedule.
- **Dependent fields:** Changing a parent field (like program or silence type) clears the dependent fields automatically.
